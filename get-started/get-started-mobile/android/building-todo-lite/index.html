Todo Liteを実際に作ってみる

Todo Liteはタスクリスト管理アプリの一つです。このアプリはドキュメントCRUD, バイナリーの添付、ビューとクエリ、認証、同期などCouchbase Liteの数多くの機能を使っています。

ターミナルを開く

ターミナルはアプリケーション/ユーティリティのフォルダにあります。

ディレクトリの移動

ToDo Liteを保存したいディレクトリに変更します。例えば：

$ cd ~/dev

サンプルアプリの複製

Todo Liteを複製しましょう:

$ git clone https://github.com/couchbaselabs/ToDoLite-Android
$ cd ToDoLite-Android
$ git submodule init && git submodule update

Android Studioへ読込み

  1. build.gradle または settings.gradle をプロジェクトのルートから選択し、Android Studioにプロジェクトをインポートしましょう。

  2. "play" または "debug" ボタンを使って、アプリを起動してみましょう。

  3. オプション： Sync Gatewayで同期する場合、app/src/main/java/com/couchbase/todolite/Application.java を開いて SYNC_URL を自分のSync Gateway URLに変更しましょう。
