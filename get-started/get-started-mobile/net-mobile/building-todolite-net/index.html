Todo Liteを作ってみる (Coming soon!)

Todo Liteはタスクリスト管理アプリの一つです。このアプリはドキュメントCRUD, バイナリーの添付、ビューとクエリ、認証、同期などCouchbase Liteの数多くの機能を使っています。アプリ作成後、実際に動かすことでソースコードを読んだり、実際に動作している状況をみることができます。

現在.NETのサンプルを開発中で、近日中に公開予定です！ しばらくの間、GrocerySyncや当社イベントCouchbase Connectカンファレンスでのアプリなどこちら.
