Creating the manager and a database

To get started with the CouchbaseEvents app code, you need to add a bit of initial code. After you have the initial code in place, you can build and run the app.

Every Couchbase Lite application needs a manager object, which manages access to databases. This section shows you how to create the manager and a database.

When following along, you add all the sample code included in this section to the MainWindow class.

Add initial code

The initial code sets up a constant, TAG, for the logging messages that mark the progress of this simple app and outputs some log messages. Add the following statements at the beginning of the MainWindow. 

public partial class MainWindow : Window
{
    const string TAG = "CouchbaseEvents";
    public MainWindow()
    {
        InitializeComponent();
        Debug.WriteLine("Begin Couchbase Events App", TAG);
        Debug.WriteLine("End Couchbase Events App", TAG);
    }
}

NOTE: For any classes that you add, you can right-click on the class and select Resolve to add the appropriate namespace. The Couchbase Lite APIs are available under the Couchbase.Lite namespace.

Build and run the app

Visual Studio writes the Debug statements to the Output window. Build and run the app, and search the Output for the string CouchbaseEvents to find the logging messages output by the CouchbaseEvents app as shown below: 

CouchbaseEvents: Begin Couchbase Events App
CouchbaseEvents: End Couchbase Events App

Events App

We are going to create an events application with Couchbase Lite. The application is based on events users would create, such as ‘party’, or a ‘rock concert’ they wish to attend. The tutorial assumes you are already familiar with WPF adnd .NET.

The MainWindow class

The MainWindow class added by the WPF template is used to invoke the HelloCBL method, which serves as the primary location where Couchbase Lite is called in this tutorial.

Add code to the MainWindow constructor as shown below:

public partial class MainWindow : Window
{
    const string DB_NAME = "couchbaseevents";
    const string TAG = "CouchbaseEvents";
    Database db;
    public MainWindow()
    {
        InitializeComponent();
        Debug.WriteLine("Begin Couchbase Events App", TAG);
        HelloCBL();
        Debug.WriteLine("End Couchbase Events App", TAG);
    }
}

Creating the database

Next, add the following implementation of HelloCBL:

void HelloCBL()
{
    try
    {
        db = Couchbase.Lite.Manager.SharedInstance.GetDatabase(DB_NAME);
    }
    catch (Exception e)
    {
        Debug.WriteLine("{0}: Error getting database: {1}", TAG, e.Message);
        return;
    }
}

The Manager.SharedInstance returns a singleton, top-level object that manages a Couchbase Lite database. We use it here to return a database named "couchbaseevents." The call to GetDatabase will create the database if it does not already exist.
