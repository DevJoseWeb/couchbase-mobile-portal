Creating the manager and a database

Every Couchbase Lite application needs a manager object, which manages access to databases. This section shows you how to create the manager and a database.

When following along, you add all the sample code included in this section to the HelloWorld MainActivity class.

Create a manager

  1. Add the following code before the last log statement in the onCreate method: 

    // create a manager
Manager manager;
try {
    manager = new Manager(new AndroidContext(this), Manager.DEFAULT_OPTIONS);
    Log.d (TAG, "Manager created");
} catch (IOException e) {
    Log.e(TAG, "Cannot create manager object");
    return;
}

    The code snippet creates a manager object and then outputs a message to the logcat. The manager is used later in the app to access the database.

  2. Build and run the app.

  3. View the console output.

    The logcat includes a message similar to the following one that says the run was successful.

    05-16 19:46:57.929  31719-31719/com.couchbase.helloworld D/HelloWorld﹕ Manager created

Create a database

  1. Add the following code before the last log statement in the onCreate method: 

    // create a name for the database and make sure the name is legal
String dbname = "hello";
if (!Manager.isValidDatabaseName(dbname)) {
    Log.e(TAG, "Bad database name");
    return;
}
// create a new database
Database database;
try {
    database = manager.getDatabase(dbname);
    Log.d (TAG, "Database created");
} catch (CouchbaseLiteException e) {
    Log.e(TAG, "Cannot get database");
    return;
}

    The code snippet sets up and validates a name for the database and then creates the database. It is important to validate the database name because it must conform to the following rules:

    • The database name must begin with a lowercase letter.
    • The database name must contain only valid characters. The following characters are valid in database names:
      • Lowercase letters: a-z
      • Numbers: 0-9
      • Special characters: _$()+-/

    The file for the database has a .cblite extension.

  2. Build and run the app.

  3. View the console output.

    The logcat includes a message similar to the following one that indicates the database was created.

    05-16 19:46:57.933  31719-31719/com.couchbase.helloworld D/HelloWorld﹕ Database created
