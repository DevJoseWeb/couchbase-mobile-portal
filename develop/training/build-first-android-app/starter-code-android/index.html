This lesson teaches you to

  1. Import Couchbase Lite packages
  2. Import Java packages
  3. Add initial code
  4. Build and run the app

Adding starter code

To get started with the HelloWorld app code, you need to import Couchbase Lite packages and add a bit of initial code. After you have the initial code in place, you can build and run the app. By building the app at this stage of development you can make sure that Couchbase Lite for Android and other dependencies were set up correctly.

Import Couchbase Lite packages

To make Couchbase Lite available to your app, you need to import some Couchbase Lite packages. Near the top of the MainActivity.java file, add the following Couchbase Lite import statements:

import com.couchbase.lite.*;
import com.couchbase.lite.android.AndroidContext;
import com.couchbase.lite.util.Log;

Import Java packages

In addition to the Couchbase Lite packages, you need to import some Java classes used by the HelloWorld app. After the Couchbase Lite imports, add the following import statements. 

import java.io.IOException;
import java.text.SimpleDateFormat;
import java.util.Calendar;
import java.util.GregorianCalendar;
import java.util.HashMap;
import java.util.Map;

Add initial code

The initial code sets up a constant, TAG, for the console logging messages that mark the progress of this simple app and outputs some log messages. Add the following statements at the end of the MainActivity class:

final String TAG = "HelloWorld";
Log.d(TAG, "Begin Hello World App");
Log.d(TAG, "End Hello World App");

Build and run the app

Build and run the app. If you filter the logcat output on the stringHelloWorld, you'll see only the logging messages output by the HelloWorld app. Look for logging messages similar to these in the logcat:

05-16 19:46:57.925  31719-31719/com.couchbase.helloworld D/HelloWorld﹕ Begin Hello World App
05-16 19:46:57.961  31719-31719/com.couchbase.helloworld D/HelloWorld﹕ End Hello World App
