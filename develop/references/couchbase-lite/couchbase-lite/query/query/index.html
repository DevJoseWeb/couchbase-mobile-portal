Enums

IndexUpdateMode
Queryが実行される時にどのタイミングでViewのインデックスが更新されるか指定するためのenum

Values

Before必要あればQueryが実行される前にインデックスが更新される（デフォルト）。取得する速度を犠牲にして最新情報を確保する方法になる。
NeverQueryを実行している時に一切インデックスを更新しない。最新情報ではないかもしれないけど、一番早く取得することを確保する。
After必要あれば、Queryが実行された後に非同期にインデックスを更新する。最新情報ではないかもしれないかもしれないけど、一番早く取得することを確保する上に次回のQueryがもっと新しい情報を取得することも確保する。
